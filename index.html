CSS Selectors & Specificity

Selector Name Description Example
Type Selector (A) Selects elements of type A div, p, and h1 are element types that can be selected
ID Selector (#id) Selects elements of type #id. #header, #first-text, and #code are id types that can be selected.
Descendant Selector (A B) Selects all B elements inside A. div p , selects all p tags in divs.
Combine descendant & id Selector (#header A) Selects all B elements inside A. div p , selects all p tags in divs.
Class Selector (.button) Selects all elements with the class attribute. div p , selects all p tags in divs.
Combine Class Selector (A.button) Selects all A elements with button class. #big.post , selects all element with id of big thaat also have a class of post.
Comma Combinator (A, B) Selects all A and B elements. div, p , selects all p's divs.
Universal Selector (*) Selects all elements. * , selects all elements and styles them accordingly.
Combine Universal Selector (A *) Selects all elements inside A. p *, selects all elements in all p tags.
Adjacent Siblings (A + B) Selects all B elements directly following A,
elements that are on the same block are regarded as siblings.		 p + div selects all p tags that are direct siblings of div.
General Siblings (A ~ B) Selects all B elements that follow A, the difference between
this and Adjacent sibblings is that Adjacent siblings select
only one sibbling that follows it directly. 		div ~ p, selects all p elements that are siblings of divs.
